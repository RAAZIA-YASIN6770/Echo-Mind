Parental Consent Mockup (textual)

Screen 1: Age Gate
- "How old are you?" [DOB entry]
- If under 13: "Please enter a parent/guardian email to continue."

Screen 2: Parent Email Confirmation
- Parent receives email with consent link: "Confirm consent for [child name] to use Eco-Mind AI"
- Email includes consent details, data collected, and contact for privacy questions.

Screen 3: Consent Confirmation
- Parent clicks link, confirms checkbox: "I give consent for my child to use Eco-Mind AI as described"
- Return to app with success message.

Backend flows: See `parental_consent.md` for API endpoints and retention policy.
